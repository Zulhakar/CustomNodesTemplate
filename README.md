# OnlyBlends.Core
It's basically a Custom Node Editor, separated from and for [OnlyBlends.Mixer](https://github.com/Zulhakar/OnlyBlendsMixer) and other projects like [OnlyBlends.Gamepad](https://github.com/Zulhakar/OnlyBlendsGamepad).

## Features

This Repo is not only the Core of OB.Mixer and OB.Gamepad or a Template for other devs to create a custom node editor, it has its own use case to spawn objects from Geometry Node Modifiers. So you can install it if you only need this feature. If you already use OB.Mixer or OB.Gamepad there is no need to install it.
