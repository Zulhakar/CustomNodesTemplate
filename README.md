# Custom Nodes Template
It's a Custom Node Editor, separated from and for [OnlyBlends.Mixer](https://github.com/Zulhakar/OnlyBlendsMixer) and other projects [OnlyBlends.Gamepad](https://github.com/Zulhakar/OnlyBlendsGamepad).

## Features

This is not only for my projects or maybe a template for devs, it has its own use case to spawn objects from Geometry Node Modifiers.
